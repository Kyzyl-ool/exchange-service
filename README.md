# Биржевой сервис

Подержан брокер Тинькофф Инвестиций. Перед запуском необходимо создать `.env`-файл и указать токен.

Для запуска введите команду:

```
python wsgi.py
```

Либо можно просто собрать Docker-образ и запустить платформу в контейнере.
